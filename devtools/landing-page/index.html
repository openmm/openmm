OpenMM

OpenMM is a high performance toolkit for molecular simulation. Use it as a library, or as an application. We include extensive language bindings for Python, C, C++, and even Fortran. The code is open source, licensed under a dual BSD/LGPL license. The source code is actively maintained on Github.

Download Github Learn more

Extreme Flexibility. Extreme Speed.

Extreme flexibility through custom forces and integrators. Extreme performance through GPU Acceleration. We support both AMD, NVIDIA, and Intel Integrated GPUs. Oh, we support the CPU too.
